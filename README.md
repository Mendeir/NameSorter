# NameSorter
A program that receives multiple inputs from user and sorts it

### Sample Input:
```
How many names would you like to enter? 5

Enter name #1: Will
Enter name #2: Tom
Enter name #3: Jerry
Enter name #4: Rick
Enter name #5: Morty
```

### Sample Output: 
```
Name #1: Jerry
Name #2: Morty
Name #3: Rick
Name #4: Tom
Name #5: Will
```
